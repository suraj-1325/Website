Projects

Hands-on cybersecurity projects demonstrating practical application of security concepts

Isolation Office LAB

Key services include Active Directory (AD) for authentication, DNS for name resolution, a Mail Server for internal communication, and a Web Server hosting applications such as DVWA and Service Website. A NAS provides centralized storage, while Veeam manages backups and vCenter administers ESXi hosts. Wazuh is deployed for log monitoring, and Kali Linux is used for security testing.

The network operates on a private IP range with static assignments for servers, and connectivity, backup, and security are rigorously tested to ensure a functional and secure environment. Additionally, OPNsense is used to provide Internet access and VPN connectivity for accessing isolated servers.

Threat Intelligence Integration in Wazuh

We integrated OpenCTI and MISP with Wazuh to enhance threat intelligence capabilities. In addition, core switch SPAN/mirror traffic is forwarded to Zeek IDS, where it is analyzed and converted into JSON-formatted logs. These logs are then sent to Wazuh for centralized monitoring and analysis of network traffic.

This integration significantly improved our threat detection capabilities by correlating internal security events with external threat intelligence, enabling faster identification of potential security incidents.
